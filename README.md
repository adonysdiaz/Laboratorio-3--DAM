App de Recetas con Electron – README

Este proyecto es una aplicación de escritorio desarrollada con Electron, que permite visualizar recetas aleatorias obtenidas desde la API pública TheMealDB, consultar detalles y guardar recetas como favoritas utilizando localStorage.

La carpeta node_modules fue eliminada antes de subir el proyecto a GitHub para reducir considerablemente el peso del repositorio.

NOTA: Hay que volver a instalar las dependencias Después de descargar o clonar este repositorio, abra una terminal dentro de la carpeta del proyecto y ejecute:
npm install
Este comando descargará automáticamente todas las dependencias necesarias para que la aplicación funcione.
Una vez finalizada la instalación de dependencias, ejecute:

npm start


Esto abrirá la ventana principal de la aplicación Electron.

 Tecnologías utilizadas
